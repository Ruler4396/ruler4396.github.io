- $y'=f(x)$是函数$y=f(x)$的一阶导数
- $y^{(n)}=f(x)$是函数$y=f(x)$的$n$阶导数
- 二阶及以上导数被称为高阶导数

### 常见的高阶导数
- $(e^x)^{(n)}=e^x$
- $(sinx)^{(n)}=sin(x+n·\cfrac{π}{2})$
- $(cosx)^{(n)}=cos(x+n·\cfrac{π}{2})$
- $(x^a)^{(n)}=a(a-1)(a-2)…(a-n+1)$
- $(x^n)^{(n)}=n!$
- $(x^n)^{(n+k)}=0$
- $(u±v)^{(n)}=u^{(n)}±v^{(n)}$
- $(uv)^{(n)}=∑^n_{k=0}C^k_nu^{(n-k)}v^k$ #莱布尼茨公式
