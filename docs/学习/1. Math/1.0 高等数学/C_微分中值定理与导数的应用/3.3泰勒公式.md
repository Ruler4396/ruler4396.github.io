#泰勒公式
$Δy≈f'(x)Δx$
$f(x)-f(x_0)≈f'(x_0)(x-x_0)$

- 一次表达式
$f(x)≈f(x_0)+f'(x)(x-x_0)$

$P_n(x)=a_0+a_1(x-x_0)+a_2(x-x_0)^2+…+a_n(x-x_0)^n+R_n(x)=f(x)$
>$R_n(x)=o((x-x_0)^n)$ 佩亚诺余项,特别小，是$a_n(x-x_0)^n$的高阶无穷小

$P^{(n)}_n(x)=n!a_n=f^{(n)}(x_0)$
- 次数低的会变成$0$，次数高的消不掉$x$，减去$x_0$变成$0$
$a_n=\cfrac{f^{(n)}(x_0)}{n!}$

### 3.3.1泰勒中值定理
$f(x)$在$x_0$处存在$n+1$阶导数，$∃x_0$的一个邻域
使得$f(x)=f(x_0)+\cfrac{f'(x_0)}{1!}(x-x_0)+\cfrac{f''^{(n)}(x_0)}{2!}(x-x_0)^2+…+\cfrac{f^{(n)}(x_0)}{n!}(x-x_0)^n$
$R_n(x)=\cfrac{f^{(n+1)}(ξ)}{(n+1)!}(x-x_0)^{n+1}$
$ξ$是介于$x_0$和$x$之间的某个值

#### #麦克劳林公式
$f(x)=f(0)+\cfrac{f'(0)}{1!}x+\cfrac{f''^{(n)}(0)}{2!}x^2+…+\cfrac{f^{(n)}(0)}{n!}x^{n}+\cfrac{f^{(n+1)}(θx)}{(n+1)!}x^{n+1}$
$0<θ<1$

### #例子 
#麦克劳林公式 
$f(x)=e^x$  $f'(0)=f''(0)=f^{(n)}(0)=1$  $f^{(n+1)}(θx)=e^{θx}$
$e^x=1+x+\cfrac{1}{2!}x^2+…+\cfrac{1}{n!}x^n+\cfrac{1}{(n+1)!}e^{θx}x^{n+1}$

### 泰勒展开和等价无穷小
$sinx=x-\cfrac{x^3}{3!}+\cfrac{x^5}{5x}-\cfrac{x^7}{7!}+(-1)^{m-1}\cfrac{x^{2m-1}}{(2m-1)}+R_{2m}(x)$
$cosx=1-\cfrac{1}{2!}x^2+\cfrac{1}{4!}x^4-\cfrac{1}{4!}x^6+…+(-1)^m\cfrac{1}{(2m)!}x^{2m}+R_{2m}(x)$
$ln(1+x)=x-\cfrac{1}{2}x^2+\cfrac{1}{3}x^3-…+(-1)^{n-1}\cfrac{1}{n}x^n+R_n(x)$
$(1+x)^α=1+αx+\cfrac{α(α-1)}{2!}x^2+…+\cfrac{α(α-1)…(α-n+1)}{n!}x^n+R_n(x)$
