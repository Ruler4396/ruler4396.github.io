# 11.1.1 概念

曲顶柱体的体积
曲顶的方程为$Z=f(x,y)$ ，是区域$D$内的==有界==函数，将区域$D$任意分成$N$个区域$\Delta\sigma_1…\Delta\sigma_i$，每$\Delta\sigma$上任取一点$(\xi,\eta)$

$lim_{\lambda→0}\sum_{i=1}^n=\delta\sigma_i\;f(\xi_i,\eta_i)⇔\iint_Df(x,y)d\sigma$
其中$\lambda$为直径的最大值，$d\sigma$为面积元素(一般用$dxdy$表示)
直径为图形中两点的最大距离

- $f(x,y)\ge0\;\mid\;\iint f(x,y)d\sigma=$体积
- $f(x,y)\le0\;\mid\;\iint f(x,y)d\sigma=-$体积

# 11.1.2 性质
1. $\iint_D[\alpha f(x,y)+\beta g(x,y)]d\sigma=\alpha\iint_Df(x,y)d\sigma+\beta\iint_Dg(x,y)d\sigma$
>
2. $D=D_1+D_2\;\iint_D f(x,y)d\sigma=\iint_{D1}f(x,y)d\sigma+\iint_{D2}f(x,y)d\sigma$
>
3. $f(x,y)\equiv1\;\iint_D1d\sigma=\sigma*1=\sigma$
4. $f(x,y)\le g(x,y)\iint_D f(x,y)d\sigma\le \iint_Dg(x,y)d\sigma$
	- $\mid\iint_Df(x,y)d\sigma\mid\le\iint_D\mid f(x,y)\mid d\sigma$
