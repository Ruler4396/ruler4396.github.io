# 11.2.1 利用直角坐标系计算二重积分
- $\int^b_a[\int^{\phi_2(x)}_{\phi_1(x)}f(x,y)dy]dx=\int_a^b\int^{\phi_2(x)}_{\phi_1(x)}f(x,y)dy$
>先$y$再$x$，从$x$轴上开始积，$y=\phi(x)$
- $\int^d_c[\int^{\phi_2{y}}_{\phi_1{y}}f(x,y)dx]dy=\int_c^d\int^{\phi_2(y)}_{\phi_1(y)}f(x,y)dx$
>先$x$再$y$，$y$轴上开始积，$x=\phi(y)$

画出积分区域
垂直轴作无数条垂线，若垂线与图形的交小于两个，则可以选用对应的

1. 积分区域为长方形
![[Pasted image 20230413150306.png|300]]
$\int^b_adx\int^d_cf(x,y)dy\;=\;\int^d_cdy\int^b_af(x,y)dx$

2. 积分区域为长方形，且$f(x,y)=f_1(x)f_2(y)$

## 例子

1. 
$\iint_Dxyd\sigma$ $y=1,x=2,y=x$
![[Pasted image 20230413082552.png|300]]
$x$型：原式$=\int_1^2dx\int^x_1xy\;dy$
$=\int^2_1(\cfrac{1}{2}xy^2|^2_1)dx=\int_1^2(\cfrac{1}{2}x^3-\cfrac{1}{2}x)dx=(\cfrac{1}{8}x^4-\cfrac{1}{4}x^2)|^2_1$
$=\cfrac{9}{8}$

$y$型：原式$=\int^2_1dy\int_y^2xy\;dx$
$=\int^2_1[(\cfrac{1}{2}x^2y|^2_y)]dy=\int^2_1(2y-\cfrac{1}{2}y^3)dy$
$=\cfrac{9}{8}$

2. 
$\iint_Dy\sqrt{1+x^2-y^2}d\sigma\;D:y=x,y=1,x=-1$ 
![[Pasted image 20230413090209.png|300]]
$y$型：原式$=\int^1_{-1}\int_1^yy\sqrt{1+x^2-y^2}dx$

$x$型：原式$=\int_{-1}^1dx\int^1_x\cfrac{1}{2}\sqrt{1+x^2-y^2}dy^2$
$=\int_{-1}^1dx(-\cfrac{1}{2})\int^1_x\sqrt{1+x^2-y^2}d(1+x^2-y^2)=\int_{-1}^1dx-\cfrac{1}{3}(1+x^2-y^2)^{\frac{3}{2}}|^1_x$
$=-\cfrac{1}{3}\int^1_{-1}(|x|^3-1)dx$ #易错点
$=\cfrac{1}{2}$
>偶次方根时，若可能定义域存在$\lt0$的部分，则需要取绝对值
>若函数为偶函数，则可以只取$\lt0$或$\gt0$的部分进行计算后，再乘以二
