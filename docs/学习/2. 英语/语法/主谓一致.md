1. 主语单复数和谓语单复数相同
2. 专有名词的复数形式 → 整体 作为单数
3. many a…/more than one + 名词 → 整体 作为单数
4. One and a half + 复数名词 → 整体 作为单数
5. 时间、钱、距离等单位 → 整体 作为单数
6. each、one、no one、some/any/no/every + body/one/thing 作限定词/主语 →  单数
7. One of + 复数名词结构 → 单数
8. 有集体概念的名词 → 整体用单数 个体用复数
9. a number of + 可数名词复数 → 泛指许多 用复数
10. the number of + 可数名词复数 → 特指数量为…… 用单数
11. and/both…and…连接单数名词作主语 → 复数
12. and连接两个相同概念，共用一个冠词 → 单数
13. and连接两个单数词，主语前有every、each、no、many a → 单数
14. a/an+单数名词+ or two 作主语 → 单数（an 抽象）
15. one or two + 复数名词 作主语 → 复数 （one 具体）
16. some of、plenty of、a lot of、lots of、most of、the rest of、all of、half of、part of、分数/百分比 + of + 名词短语作主语  → 谓语动词与of后的名词保持单复数
17. few、many、several、all、some后加名词作主语 → 用复数
18. 就近原则 
	距离谓语近的名词决定单复数
19. 就远原则
	but、except、besides、in addition to（四除）
	with、along with、together with、combined with、as well as（五和）
	including（包括）
	rather than、instead of（而不是）
20. you+are
- **The rabbit** *as well as other annimals* gets a carrot
	- *补充* 了**主语**
- 不定代词看含义
	- Some of them like me.
	- Everybody gets a carrot.
- 复数名词有单义
	- Ten years is a long time.
