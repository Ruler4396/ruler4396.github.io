### 助动词
- be is(现在进行)、was(被动)、are(you后)
- do 强调(does) 
- have 现在完成(has)

### 情态动词
与**动词原型**一起**构成**谓语
- can/could
- may/might
- will/would
- must
- should/shall/ought to
- dare/dared
- need
- used to

4. 使役动词后的宾语与作宾补的动词构成逻辑的主谓关系

8. do、does、did与动词原型连用

### 一般现在时，主语为第三人称单数
1. 人称代词他她它、他/她/它们
2. 单个人名
3. 单数可数名词
4. 不可数名词

### 谓语动词用第三人称单数
1. 动词结尾+s
2. s、x、ch、sh、o → 结尾+es
3. 辅音+y结尾 → 变y为i+es
4. have-has、be-is

### 动名词
- 名词 - 充当主语、定语、表语
### 现在分词
- 构成进行时态 与助动词be形成谓语，充当定语、表语、状语
	1. +ing
	2. 结尾e → 去e+ing
	3. 结尾ie → 变ie为y+ing
	4. 辅音+元音+辅音 → 双写结尾+ing

### 过去式
单独作谓语，表示过去某个时间发生动作

### 过去分词
- 动词的[[非谓语]]形式，表被动/完成
	1. +ed
	2. e结尾 → +d
