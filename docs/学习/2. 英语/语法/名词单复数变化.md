### 规律变化
- 直接+s
- o、s、x、ch、sh → +es（ch发/k/音除外）
- 辅音+y → y变i+es
- f/fe → f变v+es（闭口音+f除外）
- 辅音+oo → +s
- 辅音+o → +es
