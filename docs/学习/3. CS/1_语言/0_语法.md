## #条件运算符 #三目运算符
| 优先级 | 运算符 | 运算 | 结合关系 | 举例 |
| :-----: | :-----: | :-----: |  :------: |  :-----: |
|13|?:|条件运算|自右向左|a?b:c|
```
if(a >= b){
	a - b;
}
else{
	b - a;
}

//等价于

a >= b ? a - b : b - a;
```
