#数据类型

变量 类型 名称 值

## C语言
- 整数
	- `char` #字符 1字节 8bit
	- `int` 输入输出时用`%d` 一个字 寄存器大小
	- `short` 2字节
	- `long` 一个字
- `double` 双精度浮点数 输入时用`%lf` 输出时用`%f`
- `float` 单精度浮点数 输入输出时用`%f`
- `bool` 布尔值
- 指针

#### 类型转换
- 数据类型转换只是==暂时==的
- 自动转换的类型一定能强制转换，自动转换的类型不一定能强制转换
- 自动转换
	- `int`与`long`复合运算 将`int`转换为`long`再计算
	- `float`的计算 将`float`转化为`double`
	- `int`或`float`与`double`复合运算 将`int`、`float`转化为`double`再运算
	- `char`与`int`复合运算 将`char`的 #ASCII代码 与`int`进行运算
	- `char`和`short`参与运算时 先转化为`int`类型
- 强制转换
	`(数据类型)数据` 

#### #例子 
- `(double)100` 将100(默认为int)转换为==双浮点型==
-   `int sum = 103, count = 7; `
    `double average;`
    `average = (double) sum / count;` ==注意==，()的优先级高于/
    `printf("Average is %lf!\\n", average);`
