### 输出函数Print()
- `print()`
- 将括号内内容输出

#### 内容
1. 数字 
	`123、1.45`
	==输出结果==：`123、1.45`
2. '字符串'
	``'一二三'、'Hello World'``
	==输出结果==：`一二三、Hello World`
>记得加上半角单引号/双引号/三引号	
3. 有运算符的表达式
	`2+3`
	==输出结果==：`5`

#### 输出位置
- 控制台(显示器)
- 文件
	`fx=open('文件路径/文件名', 'a+')` [^1]
	`print(输出内容, file=fx)`
	`f.close()`[^2]
	==输出结果==：`文件路径/文件/输出内容`
>注意file=

#### 输出形式
- 换行
	`print(a)`
	`print(b)`
	==输出结果==：
	`a`
	`b`
- 不换行1 
	`print('a','b')`
	==输出结果==：
	`a b`
- 不换行2 在**变量**后加逗号
	`x = "a"`
	`y = "b"`
	`print x,`
	`print y,`
	==输出结果==：
	`a b`
- 不换行3
	`x = "a"`
	`y = "b"`
	`print x,y`
	==输出结果==：
	`a b`
