### 构成
**标识** 内存地址 `print(id(obj))`
**值** 具体数据 `print(obj)`
- **类型** ==数据类型== `print(type(obj))`
	- str 字符串 - `我爱你、I love you、'加上单引号的'、"加上双引号的"` 
		- 在同一行`'单引号'"双引号"`
		- 可以位于连续的不同行`"""三引号"""、'''三引号'''`
	- int 整数 - `108`
		- 二进制 0b
		- 八进制 0o
		- 十六进制 0x
	- bool 布尔值 - `ture、false`
		- 可以转化为整数
		- Ture 真 1  `print(ture+1)#2`
		- False 假 0  `print(false+1)#1`
	- float 浮点数 - `3.1415`

> 浮点数精度问题 - Decimal模块
	> `from decimal import Decimal`
	> `print(Decimal('1.1')+Decimal('2.2'))`

### 类型转换
#### 例子
`name=颜颜`
`age=19`
`print('这是'+name+,'今年'+age+'岁')` 
>将str和int类型进行连接

`print('这是'+name+'今年'+str(age)+'岁')`
> 利用str()函数，将int类型转化为str类型
