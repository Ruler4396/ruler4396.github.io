### 算术运算符
**返回整数、浮点数**
- `+` 两个数相加
- `-` 两个数相减、得到相反数
- `*` 两个数相乘？
- `/` 两个数相除
- `%` 取模，即执行一次除法，返回除法的余数
- `**` 幂，次方
- `//` 取整除，即执行一次除法，得到商，再向下取整

### 比较运算符
**返回布尔值，True和False**
- `==` 等于 比较两个对象是否相等
- `!=` 不等于 比较两个对象是否不相等
- `>` 大于 比较前者是否大于后者
- `<` 小于 比较前者是否小于后者
- `>=` 大于等于 比较前者是否大于等于后者
- `<=` 小于等于 比较前者是否小于等于后者

### 赋值运算符
**返回整数、浮点数**
`=` 赋值号 将值赋予变量
`+=` 加法赋值号 `c += a` 等价于 `c = c + a`
`-=` 减法赋值号 `c -= a` 等价于 `c = c - a`
`*=` 乘法赋值号 `c *= a` 等价于 `c = c * a`
`/=` 除法赋值号 `c /= a` 等价于 `c = c / a`
`%=` 加法赋值号 `c %= a` 等价于 `c = c % a`
`**=` 幂赋值号 `c **= a` 等价于 `c = c ** a`
`//=` 加法赋值号 `c //= a` 等价于 `c = c // a`

### 逻辑运算符！
`and` 布尔"与" if `x == False`，`x and y`返回`False`,else 返回`y`的**计算值**
`or` 布尔"或" if `x != 0`，`x or y`返回`x`的**计算值**,else 返回`y`的**计算值**
`not` 布尔"非" if `x == True`，`x not y`返回`x`的`False`,else 返回`Ture`
