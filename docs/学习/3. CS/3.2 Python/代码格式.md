#### 多行语句
新的一行作为语句的结束符
- 但是`\`可以将一行的语句分为多行显示
> total = item_one + \
> 			item_two + \
> 			item_three
#等价于
>total = item_one + item_two+item_three

- 语句中包括`[]、{}、()`时不需要使用多行连接符

#### 变量赋值
- 变量**使用前**必须赋值，将等号**右边**的值赋予等号**左边**的变量
>变量名 = 值
- 多个变量可以同时被赋值
