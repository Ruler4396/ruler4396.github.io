#### 对象
`p1 = Class()`
`#在类之外，通过类名+()创建一个对象p1，使其可以调用类里的属性，同时也是一个变量`
`p1.x = 2`
`p1.name = "Lee"`
`#通过对象名+.+属性名来把对象属性赋值`
`p1.say_Hi()`
`#对象名+.+方法名(类里的函数就叫方法)+()来调用方法，不需要参数`
`p1.say_Hi("HHH","YYY")`
`#输入的参数会直接变成方法里的参数`
- 当然也可以定义函数，类内部的函数要叫方法
- self表示类对象自身，在方法内部想访问类中的变量，必须使用self
- 没有self的变量是局部变量，函数结束后无法调用
`def say_hi(self,msg,zheshisha):`
	`print(f"Hi，我是{self.name}")`
	`print(f"Hi，我是{self.name},来自{msg},喜欢{zheshisha}")`
	`#对象中导入的对象直接可以用，多个对象利用逗号分开`
