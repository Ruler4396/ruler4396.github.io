# Java
### 类：自定义的数据类型 包括物体的所有属性

- 要在main函数里面定义 #程序框架

- 成员变量的默认值
	- int 0
	- short 0
	- byte 0
	- long 0
	- float 0.0
	- double 0.0
	- char \\u0000
	- boolean false
	- String null

```
class 类名{
//以下为成员变量
	数据修饰符1 数据类型1 属性A; //上方声明过的属性能在对象中使用
	数据修饰符2 数据类型2 属性B;
	数组类型[] 数组名;
//以下为成员方法
	 访问修饰符 返回类型 方法名(形参列表);{
	 语句;
	 return 返回值;
	}
}
```

### 成员方法

-  访问修饰符
	- public
	- protected
	- private
	- 默认(为空)

- 参数可以为基本类型或引用类型(数组)
- 方法不能嵌套定义
- 在不同类中，基本数据类型的修改不影响(成员变量相互独立)
- 在不同类中，引用数据类型的修改会影响全局的数据(数组、对象的引用等)
- 同一个类里的方法直接调用 不同的类中需要新建对象再调用
```
public void CenYan(int n){
	System.out.println("这个家里没有我的位置了"+n);
}
public void ChuoChuo()
	CenYan(10);
	System.out.println("希望美西颜天天开心");
```

- 只能返回一个值(需要返回多个值可以返回数组)

### 对象：一个具体的事物
- 对象名只是对象的名字 对象空间(数据)才是真正的对象
```
public class main{
	public static void main(String[] args){
	//先声明再创建 或者 直接创建
	类名 对象名1 = new 类名() //构造一个新对象 对象的属性来自于类中定义的属性
	对象名1.属性A = 值;
	对象名1.属性B = 值; //对象名.属性 可以视为一个变量，一同使用
	对象名1 方法名(实参列表) //调用类的方法
	}
}
```

- 内存分配机制
	- 栈：存放基本数据类型(局部变量)
	- 堆：存放对象(数组、具体的对象)
	- 方法区：常量池(常量，如字符串)、类加载信息
