Github
配置变量
`git config --global user.name "enmusubi4"`
`git config --global user.email "l3571725139@163.com"`

初始化本地仓库
`git init`

连接在线仓库
`git remote add origin 仓库地址`

例如：
`git remote add origin https://gitee.com/Ruler4396/GameTesting`

将在线仓库的代码下载下来
`git pull origin master`

---

将代码添加到缓存区
`git add ./`

添加本次上传注释
`git commit -m "注释"`

推送到仓库
`git push origin master`
