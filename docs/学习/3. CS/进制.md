### 机器数
- 字节长为8位，最高位存放符号 **正数为0 负数为1** 
>$0000$ $0011$ +3
>$1000$ $0011$ - 3
- +3和-3就是机器数的**真值** 
- $[数]_{原/反/补}$ 表示数是原码、反码、补码

### 原码
- 符号(0/1)加上真值的绝对值
>$[+1]$ = $0000$ $0001$
>$[-1]$ = $1000$ $0001$
- 取值范围为[-127,127]（$127=2^7-1$），即[1111 1111, 0111 1111]) 

### 反码
- 正数的反码 与原码相同
- 负数的反码 符号位不变，其余各位取反
>$[+1]$ = $[0000$ $0001]_原$ = $[0000$ $0001]_反$
>$[-1]$ = $[1000$ $0001]_原$ = $[1111$ $1110]_反$ 

### 补码
- 正数的补码 与原码相同
- 负数的补码 符号位不变，其余各位取反 随后加1
>$[+1]$ = $[0000$ $0001]_原$ = $[0000$ $0001]_反$ = $[0000$ $0001]_补$
>$[-1]$ = $[1000$ $0001]_原$ = $[1111$ $1110]_反$ = $[1111$ $1111]_补$
